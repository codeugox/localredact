Frequently Asked Questions

Everything you need to know about Local Redact.

What browsers are supported?
Chrome 118+, Firefox 115+, Safari 16.4+, and Edge 118+. All modern Chromium-based browsers work. Internet Explorer is not supported. The app works best on desktop — mobile browsers can load the app but the interface is optimized for larger screens.
Does it work on mobile?
The app will load on mobile browsers but is designed for desktop use. The review interface uses a sidebar layout and hover tooltips that work best with a mouse or trackpad. We recommend using a laptop or desktop for the best experience.
What about scanned or image-only PDFs?
Version 1.0 works with digital PDFs that have a text layer (most PDFs created from Word, Excel, or other software). Scanned documents and image-only PDFs are not supported yet. Image/OCR support is planned for v1.2.
How do I verify nothing is sent to a server?
Open your browser's DevTools (F12 or Cmd+Option+I), go to the Network tab, and clear it. Then process a document. You'll see zero outbound requests. The app enforces a Content Security Policy with connect-src: 'none', making it technically impossible for the app code to make network requests after the page loads.
What types of PII does it detect?
Social Security Numbers (SSN), Individual Taxpayer IDs (ITIN), Employer IDs (EIN), credit card numbers (with Luhn validation), phone numbers, email addresses, dates of birth, street addresses, city/state/ZIP, bank account numbers, routing numbers, passport numbers, ZIP codes, driver's license numbers, dollar amounts, and dates.
What if it misses something?
The review screen lets you see exactly what will be redacted before you download. Every detected item is highlighted. You can toggle items between "redact" and "keep." If the detector misses something, you'll see it's not highlighted during review. V1.1 will add AI-powered detection using a NER model for better coverage.
Can I undo a redaction after downloading?
No. Redaction is irreversible by design. The output PDF is a rasterized image — the original text is destroyed and replaced with solid black bars. This is the same approach used by professional redaction tools. Always keep your original file.
What's the file size limit?
50 MB. Most documents are well under this. If your PDF is larger, consider splitting it into sections.
What's the difference between the two modes?
"Identity Only" redacts personal identifiers (SSN, name, address, phone, email, etc.) but keeps dollar amounts visible. Use this for tax forms where you want to share financial data without personal details. "Full Redaction" redacts everything including dollar amounts.
Is my password-protected PDF safe?
Yes. The password you enter is used locally by your browser to decrypt the PDF. It's never stored or transmitted. The password is cleared from memory when you click "Start over" or close the tab.
Where is my data stored?
Nowhere. The app does not use localStorage, sessionStorage, IndexedDB, cookies, or any other browser storage. When you close the tab or click "Start over," all data is gone.
Is this open source?
Yes. The full source code is available on GitHub. You can audit every line of code, build it yourself, and host it on your own infrastructure.
